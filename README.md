# LearnSphere
### An Intelligent Multi-Modal Learning Ecosystem Powered by Generative AI

LearnSphere is an AI-powered educational ecosystem that transforms traditional learning into engaging visual experiences through automatically generated whiteboard animation videos.

The platform combines:
- AI Whiteboard Animation Generation
- AI Presentation Generation
- Adaptive Quiz Generation
- Research Assistance using RAG
- Academically Grounded Content Generation

It serves students, educators, and researchers by creating reliable, interactive, and visually engaging educational content.

## ✨ Features

- 🎬 AI-generated whiteboard animation videos
- 📊 Automated PowerPoint presentation generation
- 🧠 Adaptive quiz generation (MCQ, True/False, Short Answer)
- 🔍 Research assistant powered by RAG
- 📚 Related work & research summary generation
- ✅ Trusted academic retrieval to reduce fake references
- 🖼️ Multimodal AI learning experience

---

## 🏗️ System Modules

### 1. Whiteboard Animation Generator
Transforms educational topics into synchronized animated explanations with narration.

### 2. AI Presentation Generator
Creates professional academic presentations from prompts or PDFs.

### 3. Quiz Generation Engine
Automatically generates adaptive quizzes from educational materials.

### 4. RAG Research Assistant
Provides academically grounded answers using trusted sources and semantic retrieval.
